FAQ

Can a plumber fix central heating

A plumber can fix central heating but may not be qualified to install a new system. Hiring a qualified heating engineer to install your new system is best.

How much will it cost to install a central heating system in my house

It depends on the size of your home, the type of system you pick, and the installation and materials fees. Generally, expect to pay between £3,500 and £4,500 for a new system.

How long does it take to install a new central heating system

It usually takes one to two days to install a new central heating system.
